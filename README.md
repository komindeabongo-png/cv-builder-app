# CV Builder App

A modern web-based CV / resume builder built with React and Vite.

## Features
- Create and edit CVs in real time
- Clean, modern UI
- Fast development setup with Vite
- Component-based architecture with React

## Tech Stack
- React
- Vite
- JavaScript
- CSS
